CSS Layouts

Generelt

CSS layouts er måten man styler nettsiden på, dette kan gjøres på forskjellige måter. Den vanligste i dag er nok Flexbox, men Grid er den nyeste måten å gjøre det på. Ev en kombinasjon av de to. Det er også mange sider som er stylet opp med Float, selv om både Flexbox og Grid er bedre måter å style en side på. Før Grid og Flexbox måtte man også jukse litt for å få designet dit man ville ha det.

CSS Grid

CSS Grid blir brukt til å style nettsiden i CSS. Dette fungerer ved at man deler inn områder i såkalte grids, altså i rutenett. Man definerer også grid områder i designet CSSen, altså header er et området, section osv. Man posisjonerer deretter elementer inn i disse rutene. Man må definere hvilken vei disse skal gå, enten som kolonner eller rader. Dette er en relativt ny måte å posisjonere elementer på, men når man skjønner hvordan det fungerer så er det en logisk måte å få til layout på. Hvis man bruker Grid riktig vil man også kunne gjøre nettsiden responsivt på en enklere måte, da man redefinerer grid områdene. Når man bruker Grid styler man også i et to-dimensjonalt system, altså rader og kolonner.

CSS Flexbox

I CSS Flexbox plasserer man elementene inn i bokser for deretter å plassere boksene rundt på nettsiden. Denne fungerer en-dimensjonalt, altså enten som en rad eller kolonne. Dette er definert med flex-direction. Flex boks vil også legge seg default over hverandre (horisontalt) får å få til et mer responsivt design. Dette er mye mer brukervennlig enn tidligere måter å gjøre det på (CSS Float).

CSS Float

CSS Float er den måten som var vanligst å style nettsider på før, og det den gjorde at man kunne bestemme om et element skulle flyte til venstre eller høyre. Man vil også klarere (clear) elementene sånn at de ikke legger seg oppå hverandre. Det er viktig å kjenne til denne utdaterte måten å designe nettsider på da den fortsatt finnes på mange nettsider.
